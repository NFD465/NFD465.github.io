Pacman

The Pacman project is the first project I have done for my University assignment. The game has all the basic components of a Pacman game such as collisions, animations, sound effects, scoreboards, powerups and enemy AI. A tile map system is used to load in the level, where the program reads from a text file to get the coordinates of the walls, munchies, ghosts, etc... and renders them in the correct position. For the enemy AI, each colored ghosts have a different AI. The orange ghosts goes either in the verticle or horizontal direction and changes direction when it hits the wall, the blue ghosts randomly changes direction when it hits the wall, the red ghosts is programmed to patrol in a square box pattern or motion and the pink ghosts constantly chases Pacman with the ability to respawn and pass through walls.

Mario

For my Mario Project, SDL2 library was used. There are two levels available currently, with three different types of enemies. I managed to create a Screen Manager to switch between the Main Menu to the different levels. Sound effects and music is also present which is done using the Mix_Chunk struct and Mix_PlayChannel function. The player has to kill 8 Koopas to clear a level and enemies spawn from the pipes. I also tried to add new features such as the yellow platform, where the player can jump through it and a Green POW block that gives the play speed boost for six seconds.

3D Room

For my 3D Room Project, the OpenGl library was used to create the Scene. My project has a free moving camera which the user can control using WASD to move around, with being able to rotate. The scene contains objects rendered from .obj that is created in Blender and text files. Textures are loaded with .tga and .raw files to texture the objects. The objects and light source can be moved using user input in any coordinates.
